Installation
------------

Clone the repository.
~~~~~~~~~~~~~~~~~~~~~

.. code:: sh

   git clone https://github.com/theairbend3r/decoding-thoughts.git

Install the packages.
~~~~~~~~~~~~~~~~~~~~~

Create environment.

.. code:: sh

   conda env create -f environment.yml

Install packages.

.. code:: sh

   pip install requirements.txt
